# 🎸 SHATTERED RIFFS ROCK BAND

## 📌 Project Overview

The project is based on a website for a rock band.
The idea is to showcase the tour in several parts.

The main thing is to show the page's layout based on a guide used to create it.
It also shows information about the band and where they can be found, along with their social media links.
